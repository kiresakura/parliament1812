## 2. 蠟封開啟動畫

### 2.1 使用場景

- **抽卡時** — 每次抽取新卡牌
- **獲得獎勵時** — 開啟寶箱/禮包
- **解鎖成就時** — 成就揭曉

### 2.2 動畫分解

**階段 1：蠟封展示（0-0.3s）**
```
┌─────────────────────┐
│                     │
│    ┌───────────┐    │
│    │   信封    │    │
│    │  ┌─────┐  │    │
│    │  │ 蠟封 │  │    │  ← 蠟封完整，帶光澤
│    │  │ 🔴  │  │    │
│    │  └─────┘  │    │
│    └───────────┘    │
│                     │
└─────────────────────┘
```
- 蠟封微微發光
- 輕微呼吸效果（縮放 0.98-1.02）

**階段 2：蠟封融化（0.3-0.8s）**
```
┌─────────────────────┐
│                     │
│    ┌───────────┐    │
│    │           │    │
│    │  ~~~🔴~~~ │    │  ← 蠟封邊緣開始融化
│    │   ~~~~    │    │    熱氣上升
│    │    ~~     │    │
│    └───────────┘    │
│                     │
└─────────────────────┘
```
- 蠟封顏色從深紅 → 亮紅 → 橙紅
- 邊緣開始液化流動
- 煙霧/熱氣粒子上升
- 可加入「滋滋」聲效提示

**階段 3：信封打開（0.8-1.2s）**
```
┌─────────────────────┐
│      ╱─────╲        │  ← 信封蓋向上翻開
│     ╱       ╲       │
│    ┌─────────┐      │
│    │  ✨✨✨  │      │  ← 光芒從內部透出
│    │   ✨✨   │      │
│    │    ✨    │      │
│    └─────────┘      │
│                     │
└─────────────────────┘
```
- 信封蓋向上翻轉
- 金色光芒從內部透出
- 信封紙張有物理翻動感

**階段 4：卡牌浮現（1.2-1.5s）**
```
┌─────────────────────┐
│                     │
│    ┌───────────┐    │
│    │   ┌───┐   │    │
│    │   │ 🃏 │   │    │  ← 卡牌緩緩上升
│    │   │   │   │    │    帶金色粒子
│    │   └───┘   │    │
│    └───────────┘    │
│                     │
└─────────────────────┘
```
- 卡牌從信封中升起
- 伴隨金色光粒子
- 卡牌輕微旋轉展示

### 2.3 動畫規格

## 3. 墨水暈染效果

### 3.1 使用場景

- **卡牌發動時** — 效果從卡牌中心擴散
- **傷害顯示** — 數字從墨水中浮現
- **狀態變化** — 增益/減益效果附加

### 3.2 效果分解

**類型 A：擴散暈染（攻擊/影響類）**

```
T=0.0s        T=0.2s        T=0.4s        T=0.6s
   •            ●             ◉              ◎
  墨點         擴散          暈開          消散
```

**動畫描述**：
1. 中心出現墨點
2. 墨水向外擴散，邊緣不規則
3. 顏色逐漸淡化
4. 最終完全消散

**類型 B：流動暈染（治療/增益類）**

```
   ↗ ~~~
  ~~~  ↘
 ↙ ~~~
   ~~~  ↗
```

**動畫描述**：
1. 多條墨水絲帶從卡牌流出
2. 絲帶蜿蜒流向目標
3. 到達目標後環繞融入
4. 產生治療/增益光效

**類型 C：濺射暈染（暴擊/大傷害）**

```
    💥
   /|\\\
  / | \\
 /  |  \\
```

**動畫描述**：
1. 中心爆發
2. 墨水向四周濺射
3. 濺射點產生二次暈染
4. 整體快速消散

## 4. 燭光搖曳效果

### 4.1 使用場景

- **議會大廳場景** — 持續背景效果
- **密室場景** — 更強烈的搖曳
- **戲劇時刻** — 燭光加強閃爍

### 4.2 效果參數

**基礎搖曳**
```javascript
// 光源位置波動
x_offset = sin(time * 2.0) * 3 + sin(time * 5.0) * 1
y_offset = cos(time * 1.5) * 2 + cos(time * 4.0) * 0.5

// 亮度波動
brightness = 0.85 + sin(time * 3.0) * 0.1 + random() * 0.05

// 色溫波動
color_temp = lerp(warm_orange, bright_yellow, sin(time * 2.5) * 0.5 + 0.5)
```

### 4.3 光影規格

| 參數 | 數值 |
|------|------|
| 位移幅度 | ±3-5 px |
| 亮度範圍 | 80%-100% |
| 色溫範圍 | #FF8C00 ↔ #FFD700 |
| 循環週期 | 2-3 秒（不規則）|

### 4.4 層級效果

| 層級 | 效果 |
|------|------|
| 燭火本體 | 形狀變化 + 顏色變化 |
| 內層光暈 | 亮度脈動 |
| 外層光暈 | 大範圍柔光 |
| 投射陰影 | 隨光源移動 |

### 4.5 情緒變體

| 情緒 | 搖曳程度 | 亮度 | 用途 |
|------|----------|------|------|
| 平靜 | 微弱 | 85% | 正常遊戲 |
| 緊張 | 中等 | 75% | 投票階段 |
| 戲劇 | 劇烈 | 閃爍 | 關鍵時刻 |

## 6. 技術實現建議

### 6.1 推薦方案

| 特效 | 推薦技術 | 備註 |
|------|----------|------|
| 蠟封動畫 | Lottie / Rive | 複雜序列動畫 |
| 墨水暈染 | Shader + Canvas | 需要程序化生成 |
| 燭光搖曳 | Shader / 簡單動畫 | 性能優先 |
| 羽毛筆 | Lottie / SVG | 路徑動畫 |

### 6.2 性能考量

- **蠟封動畫**：預加載，首次抽卡可能延遲
- **墨水暈染**：Shader 比序列幀更節省記憶體
- **燭光**：使用 GPU 加速，避免 CPU 運算
- **提供關閉選項**：設定中可關閉特效

### 6.3 降級方案

| 設備性能 | 顯示效果 |
|----------|----------|
| 高 | 全特效 |
| 中 | 簡化粒子 |
| 低 | 靜態替代圖 |
