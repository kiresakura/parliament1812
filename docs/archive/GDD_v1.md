## 1. 概覽

### 1.1 遊戲名稱

**1812 國會風雲**（1812: Parliament Storm）

### 1.2 一句話描述

> **政治狼人殺 × 策略卡牌** — 在 1812 年的國會議場上，用法案、陰謀和口才，完成你的隱藏議程。

### 1.3 核心賣點

| # | 賣點 | 說明 |
|---|------|------|
| 1 | **策略卡牌 × 社交推理** | 市場上目前沒有這個交叉組合。STS 的構築深度 + Among Us 的人性博弈。 |
| 2 | **贏家不是票最多的人** | 每個人有隱藏議程。表面合作、暗中推進自己的政治目標。投票只是手段，不是目的。 |
| 3 | **歷史沈浸感** | 1812 年英國攝政時期為背景，真實歷史事件融入遊戲機制。不是歷史課，是歷史體驗。 |
| 4 | **雙模式** | 單機 Roguelike 爬屆構築（一個人也好玩）+ 多人議會博弈（社交高潮）。 |
| 5 | **20-25 分鐘一局** | 不拖泥帶水。午休能打完一局，睡前能來兩局。 |

### 1.4 對標競品

| 競品 | 我們學什麼 | 我們不學什麼 |
|------|-----------|-------------|
| **Slay the Spire** | 牌組構築 + Roguelike 爬塔留存（81% 活躍率） | 純戰鬥導向、無社交元素 |
| **Reigns** | 四維度平衡 + 刷卡決策的上手速度 | 過於簡化的策略深度 |
| **Among Us** | 社交推理的病毒傳播力 + 好友房模式 | 缺乏策略深度、重複性高 |
| **Civ / Paradox** | 歷史策略的深度感 | 動輒數小時的單局時長 |

**1812 的交叉定位：** 市場上不存在「策略卡牌 × 社交推理」的組合產品。我們佔據這個空白。

### 1.5 目標受眾

- **核心族群：** 20-35 歲，玩過 STS / Civ / Paradox / Among Us / Werewolf 任一款
- **策略玩家：** 追求構築深度、數值優化、長期留存
- **社交推理玩家：** 追求心理博弈、嘴砲快感、短期爆發力
- **歷史愛好者：** 對 19 世紀歐洲政治有興趣（加分項，非必要）

## 2. 遊戲模式

### 2.1 單機模式：國會 Roguelike

**概念：** 類似 STS 的爬塔構築，但「塔」是一屆又一屆的國會。玩家作為新任議員，從第一屆國會開始，透過辯論、結盟、立法，逐步累積政治遺產，最終抵達歷史結局。

#### 核心循環

```
選擇派系
    │
    ▼
隨機開局背景（20 種）
    │
    ▼
┌─────────────────────────┐
│  第 N 屆國會             │
│  ├─ 開局：預算年度揭示    │
│  ├─ 辯論階段：出牌        │
│  ├─ 投票階段：法案表決     │
│  ├─ 結算：四維度變化       │
│  ├─ 遺產轉化              │
│  └─ 牌組構築（獎勵選牌）   │
└─────────────────────────┘
    │
    ▼（重複 5-8 屆）
    │
    ▼
歷史結局觸發
```

#### 單機模式特色

- **每局 5-8 屆國會**，每屆 3-5 分鐘，整局 20-25 分鐘
- **隨機開局背景**：決定初始四維度狀態與特殊事件池
- **AI 議員**：3-7 個 AI 議員，各有自己的派系與行為模式
- **事件系統**：屆間隨機事件（戰爭爆發、瘟疫、貿易危機），影響下一屆的環境
- **遺產累積**：每屆通過的法案轉化為永久被動加成，影響後續所有屆
- **多結局**：10+ 種歷史結局，由四維度狀態與特殊事件觸發
- **政績簿**：記錄解鎖進度，鼓勵反覆遊玩

#### 難度系統

| 難度 | 說明 |
|------|------|
| **議員** | 基礎難度，適合學習機制 |
| **黨鞭** | AI 更聰明，事件更刁鑽 |
| **首相** | 極難，四維度容錯極低 |
| **攝政王** | 挑戰模式，每屆追加特殊限制 |

### 2.2 多人模式：政治狼人殺

**概念：** 4 人或 8 人進入一場議會，每人抽到隱藏議程。表面上大家都是議員在辯論法案，實際上每個人都有自己的秘密目標。贏家不是投票最多的人，是完成隱藏議程的人。

#### 核心循環

```
匹配/建立房間（4人 or 8人）
    │
    ▼
抽取派系 + 隱藏議程
    │
    ▼
┌──────────────────────────────┐
│  議會回合（共 4-6 回合）       │
│  ├─ 提案階段：提出法案         │
│  ├─ 辯論階段：出牌 + 發言       │
│  ├─ 結盟/密謀：私訊交涉         │
│  ├─ 投票階段：公開或匿名投票     │
│  └─ 結算：四維度變化 + 事件觸發  │
└──────────────────────────────┘
    │
    ▼（重複 4-6 回合）
    │
    ▼
議程揭示 + 勝負判定
```

#### 多人模式特色

- **隱藏議程系統**：每個玩家有 1 個主要議程 + 1 個次要議程
- **社交推理**：觀察其他玩家的出牌模式、投票傾向、發言內容，推測其真實目的
- **結盟與背叛**：可以私下結盟，但沒有任何機制保證盟約有效——背叛是合法策略
- **語音/文字辯論**：辯論階段可以自由發言，用口才說服其他玩家投票
- **卡牌博弈**：不只是嘴砲，卡牌提供實質的策略操作空間
- **AI 補位**：人數不足時 AI 填充，AI 也有隱藏議程且行為合理

#### 人數配置

| 人數 | 派系分配 | 隱藏議程數 | 建議時長 |
|------|---------|-----------|---------|
| **4 人** | 每人不同派系（含 1 個通用派系） | 4 個主要 + 4 個次要 | 20 分鐘 |
| **8 人** | 每派系 2-3 人 | 8 個主要 + 8 個次要 | 25 分鐘 |

## 3. 核心機制

### 3.1 派系系統

三大派系構成剪刀石頭布的克制關係：

```
    保守黨（防禦）
      /        \
  剋節奏     被破穩定
    /            \
自由派（連段）←─剋資源鏈─ 商行聯盟（控制）
```

#### 3.1.1 保守黨（The Royalists）— 防禦型

> _「傳統即力量。穩定即正義。」_

| 屬性 | 值 |
|------|-----|
| **核心資源** | 穩定度（Stability） |
| **遊戲風格** | 防禦、消耗、穩固盤面 |
| **難度** | ★☆☆（新手友好） |
| **容錯率** | 高 |
| **爆發力** | 低 |

**機制特色：**

- **穩定度護盾**：每屆開始時根據穩定度獲得「護盾值」，吸收來自其他派系的攻擊性法案效果
- **教會支持**：獨有資源「信仰值」，可消耗信仰值否決法案或強制延期投票
- **軍隊壓制**：當軍事維度高時，保守黨法案額外獲得投票加成
- **傳統權威**：每通過一個「保守」標籤的法案，下一屆所有「改革」標籤法案費用 +1

**核心卡牌風格：**

- 高防禦、低攻擊
- 回復/穩定類效果為主
- 中後期透過穩定度優勢壓制對手
- 被動加成多，主動出擊少

**剋制關係：**
- **剋自由派**：保守黨的穩定盤面打斷自由派的連段節奏。自由派需要多張卡 Combo，保守黨一張「否決」就能打斷整個連鎖。
- **被商行剋制**：商行聯盟的賄賂和購買選票直接繞過保守黨的正面防禦。金錢腐蝕穩定。

**派系專屬起始遺產：**
- 🏛️ **王室憲章**：每屆開始時，穩定度 ≥ 60 則額外獲得 1 行動點。

---

#### 3.1.2 自由派（The Reformists）— 連段型

> _「人民的聲音，就是法律的力量。」_

| 屬性 | 值 |
|------|-----|
| **核心資源** | 民意（Public Opinion） |
| **遊戲風格** | 連段、爆發、法案連動 |
| **難度** | ★★☆（中等） |
| **容錯率** | 中 |
| **爆發力** | 極高 |

**機制特色：**

- **民意引擎**：每張法案卡上有「民意值」，打出法案時累積民意。民意達到閾值時觸發「民意浪潮」，本回合所有法案費用 -1
- **法案連動**：特定法案之間有 Combo 關係。例如「新聞自由法」→「出版管制廢除」→「民意調查法」三連可以觸發「啟蒙運動」特殊效果
- **群眾演說**：獨有機制，消耗民意值直接影響投票結果（代表街頭抗議施壓）
- **改革加速**：每通過一個「改革」標籤法案，下一張「改革」法案費用 -1（可疊加，最多 -3）

**核心卡牌風格：**

- 低費用、多數量
- 連鎖觸發效果
- 單張弱，Combo 強
- 需要規劃出牌順序

**剋制關係：**
- **剋商行聯盟**：自由派的連段速度快過商行的資源累積。商行需要時間積攢金錢來操控，但自由派一波 Combo 就能在商行準備好之前推過關鍵法案。
- **被保守黨剋制**：保守黨的否決和延期機制直接打斷 Combo 鏈。失去連段的自由派單卡效果薄弱。

**派系專屬起始遺產：**
- 📰 **街頭小報**：每回合第一張打出的法案額外產生 +2 民意值。

---

#### 3.1.3 商行聯盟（The Merchant Guild）— 控制型

> _「每個人都有價格。問題只是多少。」_

| 屬性 | 值 |
|------|-----|
| **核心資源** | 資金（Gold） |
| **遊戲風格** | 控制、操縱、逆轉 |
| **難度** | ★★★（困難） |
| **容錯率** | 低 |
| **爆發力** | 高（逆轉型） |

**機制特色：**

- **賄賂系統**：消耗資金直接購買 AI 議員的選票。每個 AI 有「廉潔度」屬性，廉潔度越低越便宜
- **黑市交易**：獨有卡牌類型「交易卡」——不進入公開辯論，直接在暗處執行。其他玩家只能看到效果，看不到卡牌
- **經濟制裁**：可以對特定玩家發動經濟制裁，降低其下屆的行動點上限
- **引發事件**：消耗大量資金可以主動觸發特定事件（貿易戰、通貨膨脹、資源短缺），改變全局環境

**核心卡牌風格：**

- 費用高，效果強
- 直接操控類效果
- 資源管理是核心挑戰
- 逆風翻盤能力最強

**剋制關係：**
- **剋保守黨**：金錢繞過穩定度的正面防禦。賄賂讓保守黨的「否決」形同虛設（買通投票人），經濟制裁讓穩定度失去意義。
- **被自由派剋制**：自由派的連段速度不給商行積累資金的時間。民意浪潮讓賄賂無效（群眾的意志不可收買）。

**派系專屬起始遺產：**
- 💰 **東印度股份**：每屆結算時，財政維度每高於 50 一個等級，獲得 2 額外資金。

### 3.2 卡牌系統

#### 3.2.1 卡牌規模

| 類型 | 數量 |
|------|------|
| 保守黨專屬卡 | 50-70 張 |
| 自由派專屬卡 | 50-70 張 |
| 商行聯盟專屬卡 | 50-70 張 |
| 通用卡 | 50 張 |
| **總計** | **約 250-260 張** |

#### 3.2.2 卡牌類型

| 類型 | 圖示 | 說明 | 範例 |
|------|------|------|------|
| **法案卡** | 📜 | 核心卡牌。提出法案，經投票通過後生效。 | 「新聞自由法」「軍隊擴編令」 |
| **策略卡** | ⚔️ | 即時效果，不需投票。用於辯論階段干擾或輔助。 | 「揭發醜聞」「臨時休會」 |
| **人脈卡** | 🤝 | 持續效果，代表政治人脈關係。場上最多 3 張。 | 「攝政王密友」「海軍上將」 |
| **陰謀卡** | 🗡️ | 暗中使用，其他玩家看不到。多人模式專用。 | 「暗殺」「偽造文書」 |
| **交易卡** | 💰 | 商行聯盟專屬。不進入公開辯論，直接暗中執行。 | 「走私協議」「借貸合約」 |

#### 3.2.3 卡牌屬性

每張卡牌包含以下屬性：

```
┌─────────────────────────────────┐
│  [費用]              [稀有度]    │
│                                 │
│         【卡牌名稱】             │
│                                 │
│  ┌─────────────────────────┐    │
│  │                         │    │
│  │       [卡牌插圖]         │    │
│  │                         │    │
│  └─────────────────────────┘    │
│                                 │
│  類型：法案卡                    │
│  標籤：改革 / 經濟              │
│  派系：自由派                    │
│                                 │
│  效果：減少 2 財政維度，         │
│  增加 3 民意維度。               │
│  若本回合已打出「改革」標籤       │
│  法案，費用 -1。                 │
│                                 │
│  民意值：+2                      │
│  ─────────────────────────────  │
│  「自由的代價，從來不是免費。」    │
│          — 查爾斯·格雷伯爵       │
│                                 │
└─────────────────────────────────┘
```

**屬性說明：**

| 屬性 | 說明 |
|------|------|
| **費用** | 打出這張卡需要消耗的行動點（AP）。範圍 0-5。 |
| **稀有度** | 普通（白）/ 稀有（藍）/ 史詩（紫）/ 傳奇（金）。影響出現機率和強度。 |
| **類型** | 法案 / 策略 / 人脈 / 陰謀 / 交易。決定使用方式。 |
| **標籤** | 改革 / 保守 / 經濟 / 軍事 / 外交 / 教會 / 民生。用於 Combo 連動。 |
| **派系** | 所屬派系或通用。 |
| **效果** | 卡牌的實際效果文字。 |
| **民意值** | 自由派專屬屬性。打出時累積的民意值。 |
| **風味文字** | 歷史引言或虛構引言，增加沈浸感。 |

#### 3.2.4 稀有度分佈

| 稀有度 | 佔比 | 出現機率（獎勵選牌） | 強度 |
|--------|------|---------------------|------|
| **普通** | 50% | 60% | 基礎效果 |
| **稀有** | 30% | 25% | 中等效果 + 條件觸發 |
| **史詩** | 15% | 12% | 強力效果 + 複雜條件 |
| **傳奇** | 5% | 3% | 改變局勢的效果 |

#### 3.2.5 卡牌升級

- 單機模式中，特定事件或遺產可以升級卡牌
- 升級後卡牌效果增強（費用 -1 / 效果 +50% / 增加額外觸發）
- 升級卡有視覺差異（卡框加金邊 + 效果文字發光）

### 3.3 四維度系統

四個維度代表國家的整體狀態。玩家的每個決策都會影響這四個維度。任何維度降到 0 或超過 100 都會觸發特殊結局。

```
                    民意 (Public Opinion)
                         ▲
                         │
                         │
    軍事 ◄───────────────┼───────────────► 外交
  (Military)             │              (Diplomacy)
                         │
                         │
                         ▼
                    財政 (Treasury)
```

#### 3.3.1 維度詳細

| 維度 | 範圍 | 初始值 | 代表 | 過低後果 | 過高後果 |
|------|------|--------|------|---------|---------|
| **民意** | 0-100 | 50 | 人民對政府的信任度 | 革命爆發 | 民粹失控 |
| **財政** | 0-100 | 50 | 國庫充裕度 | 國家破產 | 貴族壟斷 |
| **軍事** | 0-100 | 50 | 軍隊力量與忠誠 | 外敵入侵 | 軍事政變 |
| **外交** | 0-100 | 50 | 國際關係與影響力 | 全面孤立 | 被捲入戰爭 |

#### 3.3.2 維度互動

維度之間存在聯動效應：

- **軍事 ↑ → 財政 ↓**：養軍隊要錢
- **民意 ↑ + 財政 ↓ → 外交 ↓**：民粹主義損害國際關係
- **外交 ↑ → 軍事 ↓**：和平談判削弱軍事存在的正當性
- **財政 ↑ → 民意 ↓**：增稅充盈國庫但人民不滿

這些聯動不是自動的，而是透過事件和法案效果體現。玩家需要在推動一個維度時，預判對其他維度的連鎖影響。

#### 3.3.3 維度閾值事件

| 維度 | 閾值 | 觸發事件 |
|------|------|---------|
| 民意 ≤ 20 | 街頭暴動：下屆所有法案費用 +1 |
| 民意 ≥ 80 | 民意高漲：本屆可以額外提出 1 個法案 |
| 財政 ≤ 20 | 經濟危機：下屆行動點 -2 |
| 財政 ≥ 80 | 黃金時代：所有法案效果 +50% |
| 軍事 ≤ 20 | 邊境告急：每屆結算額外 -5 外交 |
| 軍事 ≥ 80 | 軍隊驕兵：每屆結算 10% 機率觸發政變 |
| 外交 ≤ 20 | 國際制裁：每屆結算額外 -5 財政 |
| 外交 ≥ 80 | 盟約加持：每屆結算額外 +3 民意 |

---

### 3.4 政治遺產系統

**政治遺產 = STS 的遺物（Relics）。** 但更有主題性——每個遺產都是你在政治生涯中留下的印記。

#### 3.4.1 遺產來源

| 來源 | 說明 | 範例 |
|------|------|------|
| **法案通過** | 法案通過後轉化為永久被動加成 | 通過「新聞自由法」→ 獲得遺產「自由之聲」（每屆 +2 民意） |
| **派系專屬起始遺產** | 選擇派系時自動獲得 | 王室憲章 / 街頭小報 / 東印度股份 |
| **歷史事件遺產** | 特殊事件中做出選擇後獲得 | 「滑鐵盧勳章」（軍事 +5，但外交永久 -10） |
| **高風險法案遺產** | 通過高風險法案獲得，效果最強 | 「廢奴宣言」（民意 +10，財政 -15，獲得遺產「道德高地」） |
| **隱藏結局遺產** | 達成特定隱藏條件解鎖 | 「灰衣主教」（所有陰謀卡費用 -1） |

#### 3.4.2 遺產類型

| 類型 | 圖示 | 說明 |
|------|------|------|
| **被動遺產** | 🔵 | 永久生效，無需操作。如「每屆 +2 民意」 |
| **觸發遺產** | 🟡 | 達成條件時觸發。如「民意低於 30 時，自動恢復 10 民意」 |
| **代價遺產** | 🔴 | 強力效果但有持續代價。如「所有法案效果 ×2，但每屆 -5 財政」 |
| **一次性遺產** | ⚪ | 使用一次後消失。如「完全否決一個法案」 |

#### 3.4.3 遺產欄位

- 每個玩家最多持有 **12 個遺產**
- 超過 12 個時需要選擇替換
- 遺產組合產生 Synergy（如同時持有「教會之手」+「國王恩典」→ 觸發「神權聯盟」額外效果）

#### 3.4.4 風險與回報

遊戲鼓勵冒險。高風險法案（四維度變化大、可能觸發負面事件）產生的遺產比安全法案強得多：

| 法案風險等級 | 遺產品質 | 範例 |
|------------|---------|------|
| **安全** | 弱被動效果 | 「溫和改革」→ +1 民意/屆 |
| **中風險** | 中等效果 | 「關稅改革」→ +3 財政/屆，但外交 -1/屆 |
| **高風險** | 強力效果 | 「革命宣言」→ 民意轉化為行動點，但穩定度歸零 |
| **極端** | 改變遊戲規則 | 「弒君密謀」→ 解鎖隱藏派系路線 |

### 3.5 預算審計機制

**每屆國會不是在真空中發生的。你上一屆的財政決策，決定了你這一屆能做什麼。**

#### 3.5.1 行動點（Action Points, AP）

- 每屆開始時玩家獲得行動點（AP），用於打出卡牌
- **基礎 AP = 4**
- AP 受以下因素影響：

| 因素 | AP 調整 |
|------|---------|
| 財政維度 ≥ 70 | +1 |
| 財政維度 ≥ 50 | ±0 |
| 財政維度 30-49 | -1 |
| 財政維度 ≤ 29 | -2 |
| 特定遺產加成 | +1~+2 |
| 經濟制裁（被商行施加） | -1 |

#### 3.5.2 預算年度

每屆開局隨機揭示一個「預算年度」，為本屆增加特殊規則：

| 預算年度 | 效果 | 策略影響 |
|---------|------|---------|
| **豐收年** | 所有玩家 AP +1 | 鼓勵積極出牌 |
| **戰時經濟** | 軍事標籤法案費用 -1，民生標籤法案費用 +1 | 軍事派系優勢 |
| **經濟蕭條** | 所有玩家 AP -1，經濟標籤法案效果 ×2 | 經濟法案價值暴增 |
| **黃金年代** | 通過法案的遺產品質提升一級 | 鼓勵冒險 |
| **政治風暴** | 每張策略卡額外產生 +1 民意效果 | 自由派優勢 |
| **外交危機** | 外交維度每屆自動 -5 | 必須優先處理外交 |
| **瘟疫年** | 民意維度每屆自動 -3，醫療法案效果 ×3 | 特殊法案價值激增 |
| **改革浪潮** | 改革標籤法案投票自動 +2 支持票 | 自由派大優勢 |

#### 3.5.3 財政崩潰

當財政維度降至 0 時：

1. **下屆 AP 強制降為 2**（基礎 4 - 2 懲罰）
2. **觸發「國庫空虛」事件**：必須通過至少 1 個增加財政的法案，否則遊戲結束
3. **所有需要資金的卡牌無法使用**
4. **商行聯盟的賄賂系統失效**

這個機制確保玩家不能無視經濟維度。財政管理是生存的基礎。

### 3.6 隱藏議程系統（多人模式）

**這是多人模式的核心。每個玩家都有秘密目標。完成議程才是真正的勝利。**

#### 3.6.1 議程結構

每個玩家在遊戲開始時隨機抽取：
- **1 個主要議程**（完成即獲勝）
- **1 個次要議程**（完成可獲得額外分數 / 用於平局判定）

#### 3.6.2 主要議程列表

| # | 議程名稱 | 勝利條件 | 難度 |
|---|---------|---------|------|
| 1 | **推翻國王** | 民意 ≥ 80 且軍事 ≤ 30 時，提出「廢黜動議」並通過 | ★★★ |
| 2 | **壟斷貿易** | 連續 3 屆通過「經濟」標籤法案，且財政維度 ≥ 70 | ★★☆ |
| 3 | **煽動革命** | 民意 ≤ 20 且同時有 2 個維度 ≤ 30 | ★★★ |
| 4 | **維持和平** | 遊戲結束時四維度都在 40-60 之間 | ★★☆ |
| 5 | **軍事獨裁** | 軍事 ≥ 80 且通過「戒嚴令」 | ★★☆ |
| 6 | **宗教復興** | 通過 3 個「教會」標籤法案，且民意 ≥ 60 | ★★☆ |
| 7 | **外交霸權** | 外交 ≥ 90 或通過「歐洲同盟條約」 | ★★☆ |
| 8 | **暗中操控** | 在其他所有玩家都不懷疑你的情況下，主導通過 4 個法案 | ★★★ |
| 9 | **財富累積** | 個人資金（商行專屬）≥ 50 或通過 2 個「壟斷」法案 | ★★☆ |
| 10 | **改革先驅** | 通過 5 個「改革」標籤法案 | ★☆☆ |
| 11 | **叛國者** | 讓 2 個維度降至 0（國家崩潰即是你的勝利） | ★★★ |
| 12 | **和事佬** | 成功調解 3 次其他玩家之間的衝突（否決惡意法案） | ★★☆ |

#### 3.6.3 議程推理機制

玩家可以透過以下線索推測他人的議程：

- **出牌模式**：一直推軍事法案的人，可能是「軍事獨裁」議程
- **投票傾向**：總是否決經濟法案的人，可能是「煽動革命」議程
- **發言內容**：語音辯論中的措辭可能洩露意圖
- **結盟對象**：與特定派系結盟可能暗示議程方向

#### 3.6.4 投票揭示

- 遊戲結束後，所有玩家的議程公開
- 回放每個關鍵投票，分析每個人的真實動機
- 這個「真相大白」的時刻是社交推理的高潮

### 4.1 單機模式：Step by Step

```
╔══════════════════════════════════════════╗
║          單機模式完整流程                  ║
╚══════════════════════════════════════════╝

    ├─ 選擇派系（保守黨/自由派/商行聯盟）
    ├─ 獲得派系專屬起始遺產
    └─ 獲得起始牌組（10 張基礎卡）

【Step 2：隨機開局】
    ├─ 從 20 個開局背景中隨機抽取 1 個
    ├─ 開局背景決定：
    │   ├─ 初始四維度狀態（偏離標準 50 的程度）
    │   ├─ 初始 AI 議員組成
    │   └─ 第一屆特殊事件
    └─ 開局動畫播放

【Step 3：進入第 N 屆國會】（重複 5-8 屆）

    ┌─ 3a. 預算年度揭示
    │   └─ 顯示本屆的預算年度特殊規則
    │
    ├─ 3b. 抽牌階段
    │   ├─ 從牌組中抽 5 張手牌
    │   └─ 計算本屆 AP（基礎 4 ± 財政修正 ± 遺產修正）
    │
    ├─ 3c. 辯論階段（核心遊戲階段）
    │   ├─ 玩家打出卡牌（消耗 AP）
    │   ├─ AI 議員也會出牌
    │   ├─ 法案卡進入「待投票」區域
    │   ├─ 策略卡立即生效
    │   ├─ 人脈卡放置在場上（最多 3 張）
    │   └─ 辯論階段持續直到所有玩家/AI 用完 AP 或選擇結束
    │
    ├─ 3d. 投票階段
    │   ├─ 對「待投票」區域的每個法案依序投票
    │   ├─ 每個議員（玩家 + AI）投贊成或反對
    │   ├─ 計算票數 + 各種修正值
    │   ├─ 過半數通過 → 法案生效 → 四維度變化
    │   └─ 通過的法案有機會轉化為遺產
    │
    ├─ 3e. 結算階段
    │   ├─ 計算四維度最終值
    │   ├─ 檢查維度閾值事件
    │   ├─ 檢查結局觸發條件
    │   └─ 未觸發結局 → 進入構築階段
    │
    └─ 3f. 構築階段
        ├─ 從 3 張隨機卡中選 1 張加入牌組
        ├─ 可選：移除 1 張牌（需消耗資源）
        ├─ 可選：升級 1 張牌（需消耗資源）
        ├─ 觸發屆間事件（隨機事件，需做出選擇）
        └─ 進入下一屆

【Step 4：結局】
    ├─ 觸發歷史結局
    ├─ 結局動畫播放
    ├─ 顯示本局統計：
    │   ├─ 通過的法案數
    │   ├─ 獲得的遺產
    │   ├─ 四維度變化軌跡
    │   └─ 解鎖的政績簿項目
    └─ 返回主選單
```

### 4.2 多人模式：Step by Step

```
╔══════════════════════════════════════════╗
║          多人模式完整流程                  ║
╚══════════════════════════════════════════╝

【Step 1：配對 / 建立房間】
    ├─ 方式 A：建立好友房 → 分享房間碼 → 等待加入

    └─ 選擇人數模式（4 人 / 8 人）

    ├─ 選擇或隨機分配派系
    ├─ 每人隨機抽取 1 個主要議程 + 1 個次要議程
    ├─ 議程僅自己可見
    ├─ 獲得起始牌組（10 張基礎卡 + 派系卡）
    └─ 入場動畫：所有議員依次進入議會廳

【Step 3：議會回合】（重複 4-6 回合，每回合 3-4 分鐘）

    ┌─ 3a. 提案階段（30 秒）
    │   ├─ 每位玩家可從手牌中選擇 1 張法案卡放入「提案池」
    │   ├─ 提案池中的法案本回合將被辯論和投票
    │   └─ 不提案也可以（策略性放棄）
    │
    ├─ 3b. 辯論階段（60-90 秒）
    │   ├─ 計時辯論，所有玩家同時行動
    │   ├─ 打出策略卡干擾/輔助（消耗 AP）
    │   ├─ 語音/文字自由發言
    │   │   └─ 「各位，這個軍隊擴編令明明是在為某些人鋪路！」
    │   │   └─ 「我只是想保護國家安全。你怎麼知道我不是維持和平？」
    │   ├─ 觀察他人的出牌和發言，推測議程
    │   └─ 人脈卡此階段可放置
    │
    ├─ 3c. 密謀階段（45 秒）
    │   ├─ 可以向其他玩家發送私訊
    │   ├─ 「我們結盟？下一輪我支持你的法案。」
    │   ├─ 「我知道你想推翻國王。跟我合作，否則我公開你。」
    │   ├─ 陰謀卡在此階段使用
    │   ├─ 交易卡在此階段暗中執行
    │   └─ 結盟是口頭的——沒有機制保證。背叛是合法策略。
    │
    ├─ 3d. 投票階段（30 秒）
    │   ├─ 對提案池中的每個法案投票
    │   ├─ 投票方式：
    │   │   ├─ 預設：記名投票（所有人可見誰投了什麼）
    │   │   └─ 特殊卡效果可觸發「匿名投票」
    │   ├─ 賄賂在此階段結算（商行玩家可最後修改票數）
    │   └─ 過半數通過 → 法案生效
    │
    └─ 3e. 結算階段
        ├─ 四維度變化顯示
        ├─ 事件觸發（如有）
        ├─ 每位玩家獲得 1 張新牌（從 2 選 1）
        └─ 進入下一回合

【Step 4：最終回合結束 → 議程揭示】
    ├─ 所有玩家同時揭示自己的隱藏議程
    ├─ 系統判定每個議程是否完成
    ├─ 勝利排名：
    │   ├─ 完成主要議程 → 勝利
    │   ├─ 多人完成 → 比較次要議程完成度
    │   ├─ 都沒完成 → 比較議程完成進度百分比
    │   └─ 無人完成 → 依據四維度最終狀態評分
    ├─ 關鍵回放：回顧每個轉折點
    │   └─ 「第 3 回合，玩家 B 的那張『偽造文書』改變了一切...」
    ├─ 結算：ELO / 經驗值 / Battle Pass 進度
    └─ 返回大廳
```

## 5. 結局系統

### 5.1 設計理念

借鑑 Reigns 的多結局系統，但更有深度。結局不只是「死亡」，而是你的政治生涯的最終評價。有的結局是流芳百世，有的是遺臭萬年——但都是合法的結局。

### 5.2 結局類型

#### 5.2.1 維度崩潰結局（4 種）

任何維度降到 0 觸發：

| 維度 | 結局名稱 | 描述 |
|------|---------|------|
| 民意 = 0 | **法國大革命重演** | 人民起義，國會解散。你被拖上斷頭台（或逃亡海外）。 |
| 財政 = 0 | **國家破產** | 債台高築，國庫空虛。英國淪為歐洲笑柄。你被彈劾下台。 |
| 軍事 = 0 | **拿破崙入侵** | 軍隊潰敗，拿破崙渡海成功。英國成為法蘭西帝國行省。 |
| 外交 = 0 | **國際孤立** | 所有盟友離去，貿易封鎖。英國孤立於歐洲大陸之外，經濟崩潰。 |

#### 5.2.2 維度過高結局（4 種）

任何維度超過 100 觸發：

| 維度 | 結局名稱 | 描述 |
|------|---------|------|
| 民意 > 100 | **民粹狂潮** | 民意失控，暴民統治。你雖然受歡迎，但國家陷入無政府狀態。 |
| 財政 > 100 | **貴族壟斷** | 財富集中在極少數人手中。你成為寡頭的傀儡。 |
| 軍事 > 100 | **軍事政變** | 將軍們認為自己比議員更適合統治。你被軟禁。 |
| 外交 > 100 | **帝國過度擴張** | 為了維持國際承諾，國力被掏空。大英帝國在 1812 年就開始衰落。 |

#### 5.2.3 特殊事件結局（4+ 種）

| 結局名稱 | 觸發條件 | 描述 |
|---------|---------|------|
| **光榮和平** | 軍事 40-60 且外交 ≥ 70 且通過「和平條約」 | 你促成了與法國的和平。歐洲進入短暫的黃金時代。 |
| **工業革命先驅** | 財政 ≥ 60 且通過 3 個「科技」標籤法案 | 你的政策加速了工業革命。英國成為世界工廠。 |
| **廢奴英雄** | 民意 ≥ 60 且通過「廢奴法案」且財政 ≥ 40 | 你推動了人類歷史上最偉大的道德進步之一。 |
| **攝政王之影** | 與攝政王的人脈 ≥ 5 且通過 3 個「王室」標籤法案 | 你成為攝政王背後的實際掌權者。 |

#### 5.2.4 隱藏結局（2+ 種）

| 結局名稱 | 觸發條件 | 描述 |
|---------|---------|------|
| **灰衣主教** | 通過 5 個「陰謀」標籤法案 + 從未被任何法案直接影響 | 沒有人知道你的存在，但每一個決策都出自你手。歷史書上找不到你的名字，但你改變了一切。 |
| **哲人王** | 四維度全部在 45-55 之間通關 + 通過至少 8 個法案 | 完美的平衡。你既不偉大也不渺小，卻是最成功的。後世稱你為「最無聊的偉人」。 |

### 5.3 政績簿

**政績簿是結局收集系統 + 成就系統。** 鼓勵玩家反覆遊玩、嘗試不同策略。

#### 5.3.1 結構

```
╔══════════════════════════════════════╗
║              政 績 簿                ║
```

**適用時期：** DAU 1,000+

## 8. 視覺設計

### 8.1 整體美學方向

> **攝政時期的優雅 × 政治的陰暗面 × 紙質感的溫度**

| 元素 | 風格 |
|------|------|
| **色調** | 深棕、暗金、象牙白、暗紅 |
| **質感** | 羊皮紙、蠟封、木質、皮革 |
| **字體** | 襯線體（標題）+ 無襯線體（內文），英文用 Garamond 系列 |
| **動畫** | 沉穩、有重量感。不花俏但有儀式感 |
| **光影** | 燭光為主光源，暖色調陰影 |

### 8.2 入場儀式（前 3 秒）

這是玩家每次開始遊戲看到的第一個畫面。**必須在 3 秒內建立「權力感」。**

```
第 0.0 秒：全黑
第 0.3 秒：一根蠟燭在畫面中央點燃（音效：火柴劃燃）
第 0.8 秒：蠟燭光照亮一張羊皮紙，羊皮紙開始展開
第 1.2 秒：羊皮紙完全展開，露出模糊的議會廳背景
第 1.5 秒：議會廳燈光從左到右掃過（音效：腳步聲 + 椅子拉動）
第 2.0 秒：燈光完全亮起，議會廳全貌展現
第 2.5 秒：Logo「1812 國會風雲」以金箔壓印的方式浮現在羊皮紙上
第 3.0 秒：Logo 完成，微光閃爍，進入主選單
```

**關鍵：**
- 不能跳過（太短了不需要跳過）
- 每次都看（但每次都爽）
- 音效極重要：火柴聲 → 紙張聲 → 腳步聲 → 金屬壓印聲
- 給人的感覺：「你正在踏入一個權力的殿堂」

### 8.3 付費牆設計：「門縫裡的光」

**這是最關鍵的設計。** 免費版玩家在完成第一屆國會後遇到的付費牆。

```
【第一屆國會結束】
    │
    ▼
結算畫面顯示（正常）
    │
    ▼
議會廳的燈光逐漸熄滅（音效：蠟燭吹熄）
    │
    ▼
畫面幾乎全黑
    │
    ▼
畫面下方出現一道門
門縫透出溫暖的金色光芒
    │
    ▼
門縫中隱約可見下一屆的事件卡（模糊、無法辨認）
    │
    ▼
畫面凍結

沒有「購買」按鈕
沒有「$2.99」的標價
沒有「限時優惠」的橫幅

只有一道門

玩家自然而然地會去點那道門
點擊後 → 門緩緩打開 → 金光湧出 → 創辦人通行證購買介面
```

**為什麼這樣設計：**
- 傳統付費牆讓人感覺「被阻止」→ 負面情緒
- 門縫裡的光讓人感覺「被邀請」→ 好奇心
- 沒有價格標示 → 不會觸發「太貴了」的防禦反應
- 模糊的事件卡預覽 → 想知道接下來會發生什麼
- 整個過程是在遊戲世界內發生的 → 不會打破沉浸感

### 8.4 造型規格

#### 8.4.1 卡框皮膚規格

| 屬性 | 規格 |
|------|------|
| **解析度** | 512 × 720 px（卡牌尺寸） |
| **邊框寬度** | 外框 8px，內框 4px |
| **動態效果** | 靜態（免費/低價）/ 微動態（高價）/ 粒子效果（限定） |
| **色彩深度** | 32-bit RGBA |
| **檔案格式** | PNG（靜態）/ APNG 或 Sprite Sheet（動態） |

#### 8.4.2 投票動畫規格

| 屬性 | 規格 |
|------|------|
| **動畫長度** | 1.5-3.0 秒 |
| **幀率** | 30 FPS |
| **解析度** | 256 × 256 px（投票圖示區域） |
| **格式** | Sprite Sheet + Godot AnimationPlayer |

### 8.5 視覺一致性規範

```
免費外觀         付費外觀
────────────    ────────────
素雅            華麗
黑白灰          金棕紅
無動態          微動態
手寫體          刻印體
亞麻紙質感      羊皮紙質感
鐵釘固定        蠟封固定
```

**核心原則：** 免費版的玩家不會覺得自己的畫面很醜——只會覺得付費版的更好看。這是「素雅」和「華麗」的區別，不是「難看」和「好看」的區別。

| 階段 | 同步方式 | 延遲容忍 |
|------|---------|---------|
| **提案階段** | 客戶端提交提案 → 伺服器廣播 | < 500ms |
| **辯論階段** | 客戶端提交出牌 → 伺服器驗證 → 廣播 | < 200ms |
| **密謀階段** | P2P 私訊（經伺服器轉發） | < 300ms |
| **投票階段** | 客戶端提交投票 → 伺服器收齊 → 同時揭示 | < 100ms |
| **結算階段** | 伺服器計算 → 廣播結果 | < 500ms |

#### 9.4.3 斷線處理

| 場景 | 處理方式 |
|------|---------|
| **短暫斷線（< 10 秒）** | 自動重連 + 狀態同步 |
| **中等斷線（10-60 秒）** | 保留位置，AI 暫代 |
| **長時間斷線（> 60 秒）** | AI 永久替代，玩家獲得警告 |
| **惡意斷線（排位）** | 記錄為逃跑，ELO 懲罰 |
