# 隱私權政策 / Privacy Policy

## 繁體中文版

### 一、前言

感謝您使用《1812 國會風雲》（以下簡稱「本遊戲」）。本隱私權政策說明我們如何收集、使用、儲存及保護您的個人資料。使用本遊戲即表示您同意本政策的條款。

### 二、收集的資料類型

#### 2.1 帳號資料
- **Apple ID 登入資訊**：透過 Sign in with Apple 取得的用戶識別碼（不包含密碼）
- **顯示名稱**：您在遊戲中設定的暱稱
- **電子郵件**（選填）：僅在您選擇提供時收集，用於客服聯繫

#### 2.2 遊戲數據
- 遊戲進度與存檔
- 卡牌收藏記錄
- 對戰歷史與戰績
- 排行榜分數
- 成就解鎖記錄

#### 2.3 付款資訊
- **我們不直接收集任何付款資訊**。所有應用程式內購買（IAP）均由 Apple 的 App Store 或 Valve 的 Steam 平台處理。我們僅收到交易確認與購買項目記錄。

#### 2.4 裝置與技術資訊
- 裝置型號與作業系統版本
- 遊戲版本
- 當機報告與效能數據
- 網路連線狀態（WiFi / 行動數據）

#### 2.5 自動收集的資料
- 遊戲使用時間統計
- 功能使用頻率
- 匿名化的遊戲行為分析數據

### 三、資料使用目的

我們收集的資料僅用於以下目的：

| 目的 | 說明 |
|------|------|
| **提供遊戲服務** | 帳號驗證、遊戲存檔、配對系統 |
| **改善遊戲體驗** | 分析遊戲平衡性、修復錯誤、優化效能 |
| **客戶服務** | 回應您的問題與回報 |
| **安全防護** | 偵測作弊行為、防止帳號盜用 |
| **法律遵循** | 遵守適用法律法規的要求 |

**我們不會將您的資料用於廣告投放或出售給第三方。**

### 四、第三方分享

我們可能與以下第三方服務共享有限的資料：

| 第三方 | 共享資料 | 目的 |
|--------|---------|------|
| **Apple（App Store / Game Center）** | 用戶識別碼、成就、排行榜分數 | 帳號驗證、IAP 處理、Game Center 功能 |
| **Valve（Steam）** | 用戶識別碼、成就、遊戲時數 | 帳號驗證、Steam 功能整合 |
| **Google Analytics / Firebase** | 匿名化使用數據 | 遊戲分析與效能監控 |
| **伺服器託管服務商** | 遊戲數據（加密傳輸） | 提供多人遊戲伺服器 |

所有第三方均受其各自隱私權政策約束。我們要求合作夥伴採取適當的安全措施保護您的資料。

### 五、資料保留期限

| 資料類型 | 保留期限 |
|----------|---------|
| 帳號資料 | 帳號存續期間 + 刪除後 30 天 |
| 遊戲數據 | 帳號存續期間 + 刪除後 30 天 |
| 交易記錄 | 依法律要求保留（通常 5 年） |
| 當機報告 | 90 天 |
| 分析數據（匿名化） | 最長 2 年 |

帳號刪除後，我們將在 30 天內從我們的伺服器中移除您的可識別個人資料。匿名化的統計數據可能會繼續保留。

### 六、用戶權利

您享有以下權利：

#### 6.1 刪除帳號
您可以隨時透過以下方式申請刪除帳號：
- 遊戲內：設定 → 帳號 → 刪除帳號
- 電子郵件：發送刪除請求至 privacy@rush8787.com

帳號刪除後，所有相關遊戲數據將在 30 天內永久移除。此操作不可逆。

#### 6.2 資料匯出
您可以申請匯出您的個人資料副本：
- 電子郵件：發送匯出請求至 privacy@rush8787.com
- 我們將在 30 個工作天內處理您的請求

#### 6.3 資料更正
您可以在遊戲內修改您的顯示名稱。如需更正其他資料，請聯繫我們。

#### 6.4 撤回同意
您可以隨時停止使用本遊戲並刪除帳號，以撤回對資料收集的同意。

### 七、兒童隱私

本遊戲適合 12 歲以上的用戶。我們不會故意收集 12 歲以下兒童的個人資料。如果我們發現已收集到兒童資料，將立即刪除。

### 八、資料安全

我們採取以下措施保護您的資料：
- 所有網路傳輸均使用 TLS/SSL 加密
- 伺服器端資料加密儲存
- 定期安全審計
- 存取控制與權限管理

### 九、政策變更

我們可能會不定期更新本隱私權政策。重大變更將透過遊戲內通知或其他適當方式告知您。繼續使用本遊戲即表示您同意更新後的政策。

### 十、聯絡方式

如有任何隱私相關問題，請透過以下方式聯繫我們：

- **電子郵件**：privacy@rush8787.com
- **一般客服**：support@rush8787.com

## English Version

### 1. Introduction

Thank you for using Parliament 1812 (hereinafter "the Game"). This Privacy Policy explains how we collect, use, store, and protect your personal information. By using the Game, you agree to the terms of this policy.

### 2. Types of Data Collected

#### 2.1 Account Information
- **Apple ID Sign-In Data**: User identifier obtained through Sign in with Apple (does not include passwords)
- **Display Name**: Your in-game nickname
- **Email Address** (optional): Collected only when you choose to provide it, used for customer support

#### 2.2 Gameplay Data
- Game progress and save data
- Card collection records
- Match history and statistics
- Leaderboard scores
- Achievement unlock records

#### 2.3 Payment Information
- **We do not directly collect any payment information.** All in-app purchases (IAP) are processed by Apple's App Store or Valve's Steam platform. We only receive transaction confirmations and purchase item records.

#### 2.4 Device and Technical Information
- Device model and operating system version
- Game version
- Crash reports and performance data
- Network connection status (WiFi / mobile data)

#### 2.5 Automatically Collected Data
- Gameplay session duration statistics
- Feature usage frequency
- Anonymized gameplay behavior analytics

### 3. Purpose of Data Use

We use collected data solely for the following purposes:

| Purpose | Description |
|---------|-------------|
| **Provide Game Services** | Account verification, game saves, matchmaking |
| **Improve Game Experience** | Analyze game balance, fix bugs, optimize performance |
| **Customer Support** | Respond to your inquiries and reports |
| **Security** | Detect cheating, prevent account theft |
| **Legal Compliance** | Comply with applicable laws and regulations |

**We do not use your data for advertising or sell it to third parties.**

### 4. Third-Party Sharing

We may share limited data with the following third-party services:

| Third Party | Shared Data | Purpose |
|-------------|-------------|---------|
| **Apple (App Store / Game Center)** | User identifier, achievements, leaderboard scores | Account verification, IAP processing, Game Center features |
| **Valve (Steam)** | User identifier, achievements, play time | Account verification, Steam feature integration |
| **Google Analytics / Firebase** | Anonymized usage data | Game analytics and performance monitoring |
| **Server Hosting Providers** | Game data (encrypted in transit) | Multiplayer game server operation |

All third parties are bound by their respective privacy policies. We require partners to implement appropriate security measures to protect your data.

### 5. Data Retention

| Data Type | Retention Period |
|-----------|-----------------|
| Account information | Duration of account + 30 days after deletion |
| Gameplay data | Duration of account + 30 days after deletion |
| Transaction records | As required by law (typically 5 years) |
| Crash reports | 90 days |
| Analytics data (anonymized) | Up to 2 years |

After account deletion, we will remove your identifiable personal data from our servers within 30 days. Anonymized statistical data may continue to be retained.

### 6. User Rights

You have the following rights:

#### 6.1 Account Deletion
You may request account deletion at any time through:
- In-game: Settings → Account → Delete Account
- Email: Send a deletion request to privacy@rush8787.com

After account deletion, all associated game data will be permanently removed within 30 days. This action is irreversible.

#### 6.2 Data Export
You may request a copy of your personal data:
- Email: Send an export request to privacy@rush8787.com
- We will process your request within 30 business days

#### 6.3 Data Correction
You can modify your display name in-game. For other data corrections, please contact us.

#### 6.4 Withdrawal of Consent
You may stop using the Game and delete your account at any time to withdraw consent for data collection.

### 7. Children's Privacy

This Game is intended for users aged 12 and older. We do not knowingly collect personal information from children under 12. If we discover that we have collected data from a child, we will promptly delete it.

### 8. Data Security

We implement the following measures to protect your data:
- All network transmissions use TLS/SSL encryption
- Server-side data encryption at rest
- Regular security audits
- Access control and permission management

### 9. Policy Changes

We may update this Privacy Policy from time to time. Significant changes will be communicated through in-game notifications or other appropriate means. Continued use of the Game constitutes acceptance of the updated policy.

### 10. Contact Information

For any privacy-related questions, please contact us through:

- **Email**: privacy@rush8787.com
- **General Support**: support@rush8787.com
