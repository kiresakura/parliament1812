---

## 截圖規範

| 規範 | 要求 |
|------|------|
| **每組數量** | 最少 1 張，最多 **10 張** |
| **格式** | PNG 或 JPEG（無 alpha 通道） |
| **色彩空間** | sRGB 或 Display P3 |
| **方向** | Portrait（直向）或 Landscape（橫向），但同組需一致 |
| **無圓角** | 截圖不要自帶圓角，App Store 會自動裁切 |
| **無狀態列** | 建議使用乾淨的狀態列或在設計中遮蓋 |

## 建議截圖場景（10 張）

按優先順序排列：

| # | 場景 | 說明 | 重點展示 |
|---|------|------|---------|
| 1 | **主選單** | 遊戲首頁，展示整體美術風格 | 品牌印象、美術品質 |
| 2 | **角色選擇** | 議員角色選擇畫面 | 多樣性、角色設計 |

| 4 | **辯論階段** | 議員辯論互動畫面 | 獨特遊戲機制 |
| 5 | **投票階段** | 議案投票進行中 | 政治策略元素 |
| 6 | **結算畫面** | 對局結束、勝敗結果 | 成就感、數據展示 |
| 7 | **卡牌收藏** | 卡牌收藏冊 / 牌組編輯 | 收集要素、卡牌深度 |
| 8 | **排行榜** | Game Center 排行榜 | 競爭性、社交元素 |

| 10 | **多人配對** | 房間 / 配對等待畫面 | 多人遊戲體驗 |

## 截圖設計建議

### 文字疊加
- 每張截圖上方加一行**標題文字**（如「策略出牌，掌控議會」）
- 字體大小：≥ 56pt（iPhone 6.7" 基準）
- 繁中與英文各做一組
- 文字不要超過畫面 20%

### 設計風格
- 背景：使用遊戲主色調（深藍 / 金色 / 復古色系）
- 裝置外框：可選（Apple 允許但不推薦 mockup 裝置框）
- 一致性：所有截圖使用相同的邊框、字體、色調

## App 預覽影片（選填但強烈建議）

| 規格 | 要求 |
|------|------|
| **長度** | 15–30 秒 |
| **格式** | H.264, .mov 或 .mp4 |
| **解析度** | 與對應裝置截圖相同 |
| **幀率** | 30 fps |
| **音訊** | 選填，AAC 立體聲 |
| **數量** | 每個語言每個裝置最多 3 段 |

### 預覽影片建議內容
1. 0-5s：遊戲 Logo + 進入主選單
2. 5-15s：展示核心出牌與辯論機制
3. 15-25s：投票決戰 + 結算畫面
4. 25-30s：Call to action + 遊戲 Logo
